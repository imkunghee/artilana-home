# 느끼따라

홍길과 공유해서 먼저 추진해보고, 사업화 가능성을 검토하기 위한 별도 작업 폴더입니다.

## 1차 목적

- 아이디어를 먼저 작게 추진해본다.
- 실제로 반응이 있는지 확인한다.
- 사업화 가능성을 검토한다.
- ARTILANA 본체와 섞이지 않게 별도 폴더에서 관리한다.

## 현재 기준

- 이 폴더는 홍길 공유용 프로젝트 공간입니다.
- 초기 목표는 완성형 개발이 아니라, 추진 가능성 확인입니다.
- 공개/비공개 여부는 저장소 설정에서 별도로 관리해야 합니다.

## 우선 정리할 것

1. 무엇을 먼저 보여줄지
2. 누구에게 보여줄지
3. 어떤 반응을 성공 신호로 볼지
4. 사업화 가능성이 없으면 어디서 멈출지
